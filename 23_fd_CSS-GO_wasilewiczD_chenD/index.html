Grids

To play with the grid system, start from full screen and try making the screen smaller. You will observe the paragraphs shifting how they are arranged, from a 1:1 ratio to 2:1 and 1:2. The left cell is 6/9/3 and the right cell is 6/3/9. The first number in both denotes how many columns that each cell takes up out of 12 since 12 columns is the default and most optimal. So the left cell takes 6/12 and the right cell takes 6/12 when the window is large. The second number denotes what they do when the window is medium sized and the third number denotes what they do when the window is small sized. See line 50-62.

Colors

The class of button determines how the button appears. Refer to lines 58-62.

Sizing

You can also choose the size of the button by choosing a size class. Options:Tiny, Small, Large, Expanded

Other buttons

Options: Hollow, Clear The hollow class creates a button without a background but with a border present
The clear class creates a button with neither a backgound nor a border
The disabled class creates a class that you can't click on; it looks disable
